# FedRAMP Logos

This page contains logos for use by cloud service providers.

## File Types

| File Type | Use |
| -- | -- |
| svg | - Use for websites and digital interfaces<br>- Scales perfectly at any size<br>- Supports transparent backgrounds |
| png | - Use for print, email, presentations, and web when SVG isn't supported<br>- Supports transparent backgrounds |
| jpg | - Use only when you need smaller file sizes<br>- Does not support transparent backgrounds |
